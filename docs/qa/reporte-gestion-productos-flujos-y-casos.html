1. Resumen Ejecutivo

La funcionalidad de Gestion de Productos comprende el CRUD completo del catalogo de un negocio. Permite a usuarios con rol BusinessAdmin o PlatformAdmin listar, crear, editar, eliminar y gestionar el estado y stock de productos. Este reporte documenta 52 casos de prueba con:

  • Diagramas de flujo Mermaid por cada operacion CRUD
  • Guion de video paso a paso para cada caso de prueba
  • Criterios de evidencia visual que debe capturar el video
  • Narracion (texto/audio) de cada paso durante la grabacion

4. Flujo 1 — Listar Productos con Filtros

4.1 Descripcion funcional PO

El BusinessAdmin navega a /business/products y ve la lista completa de productos de su negocio. Puede filtrar por categoria usando chips (FilterChip). Si no hay negocio seleccionado, ve un mensaje de error. Si no tiene rol autorizado, se le niega el acceso.

Precondiciones: Login como BusinessAdmin o PlatformAdmin | Negocio seleccionado | Conexion a internet

5. Flujo 2 — Crear Producto

5.1 Descripcion funcional PO

Desde la lista de productos, el admin toca "Agregar producto". Se limpia el ProductEditorStore y navega a /business/products/form en modo Create. Debe completar nombre, precio, unidad y seleccionar categoria. Puede opcionalmente agregar descripcion, cambiar estado (Draft/Published), disponibilidad y stock. Al guardar exitosamente, navega de vuelta a la lista.

Precondiciones: Login BusinessAdmin/PlatformAdmin | Negocio seleccionado | Al menos 1 categoria creada

6. Flujo 3 — Editar Producto / Draft

6.1 Descripcion funcional PO

El admin toca un producto en la lista, lo que guarda un ProductDraft en el ProductEditorStore (singleton compartido) y navega al formulario en modo Edit. El draft pre-carga todos los campos. El admin puede modificar cualquier campo y guardar via PUT /{biz}/products/{id}. Si el producto no se encuentra en la carga lazy (ensureProductLoaded), se recarga desde la API.

Precondiciones: Login BusinessAdmin/PlatformAdmin | Negocio con productos existentes | Al menos 1 categoria

6.2 Diagrama de flujo

flowchart TD

  LIST -->|"Tap en producto"| DRAFT["ProductEditorStore.setDraft(item)"]
  DRAFT --> FORM["ProductFormScreen
mode = Edit"]

  APPLY --> ENSURE["ensureProductLoaded

  ENSURE --> CATS["loadCategories(businessId)"]
  CATS --> READY["Formulario listo para edicion"]
  READY --> MODIFY["Admin modifica campos"]
  MODIFY --> SAVE["Tap: Guardar"]
  SAVE --> VAL{"Validacion
Konform"}

  VAL -->|"Valido"| HTTP["PUT /{biz}/products/{id}
ProductRequest"]
  HTTP -->|"OK"| SUCCESS["Snackbar: 'Producto guardado'

  HTTP -->|"Error"| E_SERVER["Snackbar: error"]
