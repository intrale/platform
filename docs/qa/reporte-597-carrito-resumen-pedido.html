1. Resumen Ejecutivo

La historia #597 implementa la pantalla de carrito de compras para el cliente final. Permite revisar los productos seleccionados, ajustar cantidades, ver el resumen de totales, seleccionar dirección de entrega y medio de pago, y avanzar al siguiente paso del flujo de compra.

3. Flujo de Navegación al Carrito

Descripción funcional

Desde el Home del cliente, el ícono de carrito en el encabezado superior derecho muestra un badge con la cantidad de ítems. Al tocarlo, navega a la pantalla Carrito. Si no hay ítems, el badge no se muestra.

Precondiciones:
  • Usuario logueado como cliente
  • Negocio seleccionado en sesión
  • Pantalla Home visible
flowchart TD HOME[Home del Cliente] --> CHECK_CART{¿Hay ítems
en carrito?} CHECK_CART -- Sí --> BADGE[Mostrar badge
con cantidad] CHECK_CART -- No --> NO_BADGE[Ícono sin badge] BADGE --> TAP_ICON[Tap en ícono carrito] NO_BADGE --> TAP_ICON TAP_ICON --> NAV[navigate /client/cart] NAV --> CART_SCREEN[ClientCartScreen] CART_SCREEN --> SHOW_ITEMS{¿Items en carrito?} SHOW_ITEMS -- Sí --> ITEMS_VIEW[Vista con ítems] SHOW_ITEMS -- No --> EMPTY_VIEW[Estado vacío]

4. Flujo Principal: Carrito con Ítems

Descripción funcional

Cuando el carrito tiene productos, la pantalla muestra: encabezado (título + subtítulo), lista de cards de productos, card de dirección de entrega, card de medio de pago, card de resumen de totales, y botones de acción.

flowchart TD ENTER[Entrar a Carrito] --> LOAD_DATA[LaunchedEffect: cargar direcciones y métodos de pago] LOAD_DATA --> SHOW_LIST[LazyColumn: header + ítems + dirección + pago + resumen + acciones] SHOW_LIST --> HEADER[Header: título 'Carrito' + subtítulo] SHOW_LIST --> ITEMS[Cards de productos por ítem] ITEMS --> ITEM_CARD[Thumbnail + Nombre + Precio + Subtotal + Controles] SHOW_LIST --> DELIVERY[Card Dirección de Entrega] SHOW_LIST --> PAYMENT[Card Medio de Pago] SHOW_LIST --> SUMMARY[Card Resumen: Subtotal + Envío + Total] SHOW_LIST --> ACTIONS[Botón 'Continuar pedido' + TextButton 'Vaciar carrito']

Cálculo de totales

Subtotal: Σ (unitPrice × quantity) por cada ítem
Envío estimado: 0.0 (placeholder por ahora)
Total: Subtotal + Envío estimado

5. Flujo: Controles de Cantidad

Descripción funcional

Cada card de producto tiene botones + y - y un ícono de basura. El botón - está deshabilitado cuando quantity = 1 (no baja de 1). El ícono de basura elimina el ítem directamente.

flowchart TD ITEM[Card de Producto] --> PLUS_BTN[Tap +] ITEM --> MINUS_BTN[Tap - (enabled si qty > 1)] ITEM --> DELETE_BTN[Tap ícono basura] PLUS_BTN --> INC[ClientCartStore.increment(id)] INC --> RECALC[Recalcular subtotal y total] MINUS_BTN --> DEC[ClientCartStore.decrement(id)] DEC --> CHECK_QTY{qty resultado} CHECK_QTY -- "≥ 1" --> RECALC CHECK_QTY -- "0 (qty era 1)" --> REMOVE_ITEM[Eliminar ítem del mapa] DELETE_BTN --> REMOVE[ClientCartStore.remove(id)] REMOVE --> CHECK_EMPTY{¿Carrito vacío?} CHECK_EMPTY -- Sí --> EMPTY_STATE[Mostrar estado vacío] CHECK_EMPTY -- No --> RECALC RECALC --> UPDATE_UI[Actualizar UI vía StateFlow]

6. Flujo: Vaciar Carrito

Descripción funcional

El TextButton "Vaciar carrito" muestra un AlertDialog de confirmación. El usuario puede cancelar (carrito intacto) o confirmar (carrito se vacía, se muestra estado vacío con snackbar).

flowchart TD CLEAR_BTN[Tap 'Vaciar carrito'] --> DIALOG[AlertDialog: '¿Querés vaciar el carrito?'] DIALOG --> CANCEL_BTN[Tap 'Cancelar'] DIALOG --> CONFIRM_BTN[Tap 'Vaciar'] CANCEL_BTN --> DISMISS[Cerrar diálogo, carrito sin cambios] CONFIRM_BTN --> STORE_CLEAR[ClientCartStore.clear()] STORE_CLEAR --> SNACKBAR[Snackbar: 'Tu carrito está vacío.'] STORE_CLEAR --> EMPTY_STATE[LazyColumn reemplazado por ClientCartEmptyState]

7. Flujo: Estado Vacío

Descripción funcional

Cuando ClientCartStore.items está vacío (ya sea al entrar sin productos o tras vaciar), se muestra el estado vacío centrado: ícono de carrito, mensaje principal, subtítulo, y botón "Volver al inicio".

flowchart TD CHECK[itemsList.isEmpty()] -- true --> EMPTY[ClientCartEmptyState] EMPTY --> ICON[Ícono ShoppingCart en círculo] EMPTY --> TITLE[Texto: 'Tu carrito está vacío.'] EMPTY --> SUBTITLE[Texto: 'Agregá productos desde la pantalla de Inicio.'] EMPTY --> BACK_BTN[Botón 'Volver al inicio'] BACK_BTN --> NAVIGATE{goBack() exitoso?} NAVIGATE -- Sí --> POP[Volver a pantalla anterior] NAVIGATE -- No --> HOME[navigate(/client/home)]
