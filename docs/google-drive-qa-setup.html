Configuracion Google Drive para Videos QA (OAuth)

Issue #1805 — Usa tu cuenta personal de Google (no requiere Google Workspace)

NOTA: Service Accounts no tienen cuota de almacenamiento en Drive personal. Esta guia usa OAuth 2.0 con tu cuenta, que SI tiene los 15GB gratuitos.

Paso a paso

1Crear proyecto en Google Cloud Console

Ir a console.cloud.google.com/projectcreate

Nombre: intrale-qa y click "Crear"

2Habilitar Google Drive API

Ir a APIs > Google Drive API y click "Habilitar"

3Configurar pantalla de consentimiento OAuth
  1. Ir a OAuth consent screen
  2. Tipo: Externo (para cuentas personales)
  3. Nombre de app: Intrale QA
  4. Email de soporte: tu email
  5. Scopes: agregar https://www.googleapis.com/auth/drive.file
  6. Test users: agregar tu email
  7. Guardar
4Crear credenciales OAuth
  1. Ir a Credenciales
  2. Click "Crear credenciales" > "ID de cliente OAuth"
  3. Tipo: App de escritorio (Desktop app)
  4. Nombre: intrale-qa-cli
  5. Click "Crear"
  6. Copiar Client ID y Client Secret
5Ejecutar script de autorizacion

En la terminal de Claude Code, ejecuta:

! node scripts/google-drive-oauth-setup.js <CLIENT_ID> <CLIENT_SECRET>

El script va a:

  1. Mostrarte una URL — abrila en el browser
  2. Autorizas con tu cuenta Google
  3. Google te da un codigo — copialo
  4. Pegalo en la terminal
  5. El script guarda el refresh_token en telegram-config.json
6Crear carpeta en Google Drive
  1. Ir a drive.google.com
  2. Crear carpeta: Intrale QA
  3. Copiar el ID de la URL: https://drive.google.com/drive/folders/1AbCdEfG...
  4. Agregar a .claude/hooks/telegram-config.json:
{
  "google_drive_folder_id": "1AbCdEfG..."
}
7Verificar 
node qa/scripts/qa-video-share.js \
  --issue 9999 --title "Test upload" \
  --sprint "SPR-TEST" \
  --videos "qa/evidence/1781/qa-video-final.mp4" \
  --verdict "APROBADO" --passed 1 --total 1

Resultado en telegram-config.json

{
  "bot_token": "...",
  "chat_id": "...",
  "google_oauth_client_id": "123456-abc.apps.googleusercontent.com",
  "google_oauth_client_secret": "GOCSPX-...",
  "google_oauth_refresh_token": "1//0abc...",
  "google_drive_folder_id": "1AbCdEfG..."
}
