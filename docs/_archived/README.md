## Política de archivado

- Los documentos acá se conservan por **valor histórico y de trazabilidad/auditoría**, no como referencia operativa.
- Cada archivo lleva un **banner de obsolescencia** al inicio que remite a la doc viva.
- No se editan para mantenerlos al día: si un tema sigue vigente, su documentación vive fuera de `_archived/`.
- No agregar links *entrantes* desde docs vivas hacia esta carpeta salvo para señalar explícitamente que el material es histórico.
