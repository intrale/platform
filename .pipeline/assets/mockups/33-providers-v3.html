🗝 Cómo rotar / setear una key

1. Abrí una terminal Windows (PowerShell o cmd) — no Telegram.
2. Editá el archivo de credenciales con tu editor preferido: 
notepad %USERPROFILE%\.claude\secrets\credentials.json
3. Modificá el path canónico del provider, por ejemplo providers.openai.api_key: 
{
  "providers": {
    "openai": { "api_key": "sk-proj-<NUEVA_KEY>" }
  }
}
4. Guardá. Volvé al dashboard y clickeá Reload en la card del provider (o Reload todo) para re-hidratar.
